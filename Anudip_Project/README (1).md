# 💰 Financial Loan Data Analysis

## 📌 Project Overview
This project analyzes a financial loan dataset to uncover insights about borrower profiles, loan distribution, and repayment behavior. The goal is to identify patterns that help in better credit risk assessment and decision-making.

---

## 📊 Dataset Information
The dataset includes key financial and customer-related attributes:

- 👤 Borrower Details: Income, employment status  
- 💵 Loan Details: Loan amount, term, interest rate  
- 📉 Credit Metrics: Debt-to-income (DTI), total accounts  
- 📌 Loan Status: Fully Paid, Charged Off, Current  

---

## 🎯 Objectives
- Analyze loan distribution across different categories  
- Identify factors affecting loan repayment  
- Detect high-risk customers  
- Support data-driven financial decisions  

---

## 🛠️ Tech Stack
- 📊 Microsoft Excel (Data Cleaning & Analysis)  
- 🗄️ MS SQL Server (Querying & Data Processing)  
- 📈 Tableau (Data Visualization & Dashboard)  

---

## 🔍 Key Insights
- Loan approval trends vary based on income and credit profile  
- Higher DTI ratios indicate increased risk  
- Certain loan purposes show higher default rates  
- Interest rates impact repayment behavior
