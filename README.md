# Financial Loan Dataset Analysis

## 📌 Overview
This project focuses on analyzing a financial loan dataset to understand borrower behavior, loan distribution, and repayment patterns. The dataset includes key attributes related to applicants, loan details, and financial performance.

## 📊 Dataset Description
The dataset contains the following types of information:
- Borrower details (income, employment, etc.)
- Loan information (amount, term, interest rate)
- Credit metrics (DTI ratio, total accounts)
- Loan status and repayment details

## 🎯 Objectives
- Analyze loan approval and distribution trends
- Identify factors affecting loan repayment
- Detect high-risk borrowers
- Support data-driven financial decisions

## 🛠️ Tools & Technologies
- Microsoft Excel
- SQL (MS SQL Server)
- Tableau (for visualization)

## 🔍 Key Analysis Performed
- Loan amount distribution by category
- Interest rate comparison
- Income vs loan approval analysis
- Loan status (Fully Paid, Charged Off, etc.)
- Risk segmentation based on credit factors

## 📈 Use Cases
- Credit risk assessment
- Financial reporting
- Customer segmentation
- Predictive modeling

## 📂 File Structure
- `financial_loan.csv` → Raw dataset
- `queries.sql` → SQL analysis queries (if applicable)
- `dashboard.twbx` → Tableau dashboard (if created)

## 🚀 How to Use
1. Load the dataset into Excel / SQL Server
2. Clean and preprocess the data
3. Perform analysis using SQL or Excel
4. Visualize insights using Tableau

## 📌 Conclusion
This project helps in understanding loan behavior and improving decision-making in financial institutions by identifying trends and risks.
